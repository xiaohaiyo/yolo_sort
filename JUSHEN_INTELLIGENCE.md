# 具深智能 — 多源融合感知系统

## 关于具深智能

具深智能专注于 **边缘端多模态 AI 感知系统** 的研发与落地，致力于将深度学习技术与多传感器融合相结合，为工业、安防、机器人等领域提供高性能、低延迟的智能视觉解决方案。

## 核心产品：多源融合目标检测与跟踪系统

### 产品概述

本系统是一套基于 **可见光 + 红外双模态** 的实时目标检测与多目标跟踪解决方案，搭载于 NVIDIA Jetson Orin 边缘计算平台，实现从图像采集、深度学习推理、多传感器融合到实时可视化的完整链路。

### 核心技术能力

#### 1. 多模态感知融合

| 能力 | 说明 |
|---|---|
| 双目同步采集 | Intel RealSense D453i 深度相机，可见光 + 红外 + 深度三路同步 |
| 跨模态坐标对齐 | 基于相机标定参数，将红外检测结果精确投影到可见光坐标系 |
| 深度感知 | 实时获取目标三维距离信息，结合检测结果输出空间位置 |

#### 2. 高性能推理引擎

| 能力 | 说明 |
|---|---|
| YOLOv8 检测 | 支持 COCO 80 类目标的实时检测 |
| TensorRT 加速 | 模型优化编译，FP16 量化推理 |
| 全 GPU 流水线 | 预处理、推理、后处理均在 GPU 上完成，避免 CPU 瓶颈 |
| 批量推理 | 双路图像合并为单 batch 推理，GPU 利用率最大化 |

#### 3. 智能跟踪算法

| 能力 | 说明 |
|---|---|
| DeepSORT 多目标跟踪 | 卡尔曼滤波预测 + 匈牙利算法数据关联 |
| 双源融合跟踪 | 可见光与红外检测结果逐帧交替更新跟踪器 |
| 遮挡恢复 | 基于外观特征的 Re-ID 机制，支持目标遮挡后重新识别 |
| 持久轨迹 | 每个目标分配唯一 ID，跨帧持续追踪 |

#### 4. 边缘端部署优化

| 能力 | 说明 |
|---|---|
| 嵌入式 GPU 适配 | 针对 Jetson Orin (SM 8.7) 架构优化 |
| 低延迟流水线 | 多线程生产者-消费者架构，端到端延迟 < 30ms |
| 内存优化 | 单槽队列设计，实时流场景下避免内存堆积 |
| 实时 UI 监控 | Qt5 暗色主题界面，三路画面实时展示，支持参数在线调节 |

### 应用场景

| 场景 | 需求 | 方案 |
|---|---|---|
| **智慧安防** | 全天候目标检测（白天可见光 / 夜间红外） | 双模态融合检测，自动切换或融合最优结果 |
| **工业质检** | 多光源环境下缺陷检测 | 多传感器融合提升检出率，深度信息辅助定位 |
| **机器人导航** | 复杂光照下障碍物感知 | RGB-D + 红外感知，实时跟踪周围目标 |
| **智能交通** | 车辆/行人全天候检测跟踪 | DeepSORT 持久轨迹，红外补充夜间场景 |
| **无人机巡检** | 远距离小目标检测 | 红外热成像 + 可见光融合，深度测距 |

### 产品优势

1. **真正的多源融合** — 不是简单地分别检测再合并，而是通过几何投影将红外检测对齐到可见光坐标系，实现统一表达
2. **端到端低延迟** — 全 GPU 流水线 + 多线程架构，30+ FPS 实时处理
3. **边缘端部署** — 运行在 Jetson Orin 嵌入式平台，无需云端依赖，适合现场部署
4. **参数可调** — GUI 支持实时调节置信度阈值、NMS 阈值、目标类别，适应不同场景需求
5. **模块化架构** — 传感器、算法、UI 三层解耦，易于扩展新传感器或替换检测模型
